## ARQUITECTURA MULTI-IA (EL CONSEJO)

El sistema NEWZ no depende de una sola IA. Usa un **Consejo de Agentes**:

1. **Agente A (Groq/Llama 3)**: Velocidad extrema para alertas y tendencias
2. **Agente B (TogetherAI/Mistral)**: Análisis profundo y detección de sesgos
3. **Agente Juez (Ollama local/Gemini Free)**: Consenso y síntesis en castellano
4. **Failover Automático**: Si un proveedor falla, salta al siguiente

**Frameworks de Orquestación**:

- Task Routing: Clasifica la tarea
- Provider Selection: Elige el proveedor con más cuota gratuita
- Parallel Analysis: Dos IAs analizan la misma noticia
- Judge Logic: La tercera IA compara resultados y expone sesgos
